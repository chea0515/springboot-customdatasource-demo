1.
pom.xml添加(springboot要使用1.x版本)：
<dependency>
	<groupId>org.springframework.boot</groupId>
	<artifactId>spring-boot-starter-aop</artifactId>
	<version>1.5.9.RELEASE</version>
</dependency>

2.
在启动类加上注解：@Import(DynamicDataSourceRegister.class)

3.配置文件(application.yml)添加数据源：
my.datasource:
  names: datasource1, datasource2, datasource3 # 更多...
  datasource1:
    driverClassName: ...
	url: ...
	username: ...
	password: ...
  datasource2:
    driverClassName: ...
	url: ...
	username: ...
	password: ...
  # 更多...
  
4.
使用方法：在需要切换数据源的方法或者类上添加注解：@TargetDataSource("datasource1")
